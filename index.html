Stefan Häusler and Wolfgang Maass, 2006

    The package HaeuslerMaass06.zip contains software used for the simulations of the article A Statistical Analysis of Information-Processing Properties of Lamina-Specific Cortical Microcircuit Models. Cereb Cortex, Advance Access published on February 15, 2006.

    Installation

    Unpack the zip file in a directory of your choice and add this directory to your Matlab search path. This package contains only csim MEX files for Linux.

    Software description

    The code is self-contained and the matlab executable do_simulate.m is located in the folder lsm/research/HaeuslerMaass06/.

    Have fun!
